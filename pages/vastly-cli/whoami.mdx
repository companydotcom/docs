# vastly <em> whoami</em>
import { Callout } from 'nextra-theme-docs'

The `vastly whoami` command is used to show the username (email) and session token of the user currently logged into [Vastly CLI](/vastly-cli).

<br/>

# Usage
  ```bash copy filename="terminal"
  vastly whoami
  ```

  Use the `--token` flag to return only the current token
    ```bash copy filename="terminal"
  vastly whoami --token
  ```
  ## Global Options
  The following options can be passed when using `vastly whoami` command:
  - `--token`
  - `-t`
