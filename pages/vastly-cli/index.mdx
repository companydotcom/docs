# Get Started

Learn how to use the Vastly command-line interface (CLI) to configure and manage your Vastly account
and infrastructure from the command line.

## Vastly CLI Overview

import { Tab, Tabs } from "nextra-theme-docs"
import { Text, SimpleGrid } from "@vastly/ui"

With the command-line interface (CLI) provided through
[`create-wave-app`](https://www.npmjs.com/package/create-wave-app), you can interact with the
backend of your application using your terminal. This enables you to [login](/cli/login) and
[logout](/cli/logout) of your Vastly account, store and manage [env](/cli/env) variables, and more!

## Installing Vastly CLI

Vastly CLI is included as a dependency of `create-wave-app`, but if you need to install or update:

<Tabs items={["npm", "yarn"]}>
  <Tab>
  ```bash copy filename="terminal"
  npx i -g @vastly/cli
  ```
  </Tab>
  <Tab>
  ```bash copy filename="terminal"
  yarn global add @vastly/cli
  ```
   </Tab>
</Tabs>
