# Get Started
Learn how to use the Vastly command-line interface (CLI) to configure and manage your Wave App from the command line.

## Vastly CLI Overview
import { Tab, Tabs } from 'nextra-theme-docs'
import { HStack, Button, Link, SimpleGrid } from '@vastly/ui'

With the command-line interface (CLI) provided through `create-wave-app`, you can interact with the backend of your application using your terminal. This enables you to <Link href="/cli/login"><u>login</u></Link> and <Link href="/cli/logout"><u>logout</u></Link> of your Vastly account, store and manage <Link href="/cli/env"><u>secrets</u></Link>, <Link href="/cli/generate"><u>generate</u></Link> a microservice or CIAM policies, and more!

## Installing Vastly CLI
Vastly CLI is included as a dependency of `create-wave-app`, but if you need to install or update:

  <Tabs items={['pnpm', 'npm', 'yarn']}>
    <Tab>
      ```jsx copy
      pnpm i -g @vastly/cli
      ```
      </Tab>
    <Tab>
      ```jsx copy
      npm i -g @vastly/cli
      ```
    </Tab>
    <Tab>
      ```jsx copy
      yarn global add @vastly/cli
      ```
    </Tab>
  </Tabs>
