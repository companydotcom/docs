# vastly <em> whoami</em>
import { Callout } from 'nextra-theme-docs'

The `vastly whoami` command is used to show the username (email) and session token of the user currently logged into [Vastly CLI](/cli).

<Callout>
</Callout>

<br/>

# Usage
<Callout type="warning">
You will not be able to login unless you've signed up for a Vastly account
</Callout>
  ```bash copy filename="terminal"
  vastly whoami
  ```
  ## Global Options
  The following options can be passed when using `vastly whoami` command:
  - `--token`
  - `-t`
