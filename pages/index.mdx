# Wave Docs

import { Callout, Tabs, Tab } from 'nextra-theme-docs'
import { LinkCards, LinkCard } from '@components/link-card'

Wave is a modern service to help build full-stack React applications on AWS. Featuring end-to-end
typesafety, you can build and ship fast without breaking anything.

<Callout type="warning" emoji="⚠️">
  Note that to properly deploy your Wave app, and access certain features, you need a verified
  Vastly account previously set up by one of our team members. Contact us if you are having
  problems.
</Callout>
