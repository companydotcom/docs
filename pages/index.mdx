## Quickstart

To create a new Wave monorepo, use our create-wave-app npm package:

<Tabs items={['pnpm', 'npm']}>
  <Tab>
    ```bash
    pnpx create-wave-app
    ```
  </Tab>
  <Tab>
    ```bash
    npx create-wave-app
    ```
  </Tab>
</Tabs>
